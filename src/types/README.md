# Thư mục Types (`src/types/`)

Thư mục này chứa các khai báo kiểu dữ liệu hoặc cấu trúc chung của ứng dụng bằng TypeScript.

## Nội dung chính
- Định nghĩa các interfaces, types, enums dùng chung cho toàn bộ dự án để đảm bảo an toàn kiểu (type safety).

## Ví dụ cấu trúc
```text
src/types/
├── index.ts           # Export các định nghĩa kiểu dữ liệu dùng chung
├── vocab.ts           # Khai báo cấu trúc đối tượng từ vựng, flashcard
└── user.ts            # Khai báo cấu trúc dữ liệu tài khoản
```
