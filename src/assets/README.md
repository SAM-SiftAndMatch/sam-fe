# Thư mục Assets (`src/assets/`)

Thư mục này dùng để chứa các tài nguyên tĩnh (static assets) được sử dụng trực tiếp trong ứng dụng React và được đóng gói bởi Vite.

## Nội dung chính
- **Hình ảnh & Icons**: File ảnh (`.png`, `.jpg`, `.svg`), favicon, ảnh minh họa.
- **Fonts**: Các bộ font chữ custom tải về nếu không sử dụng CDN.
- **Styles/Static Files**: Các file tài nguyên không thay đổi theo thời gian.

## Ví dụ cấu trúc
```text
src/assets/
├── images/
│   ├── logo.svg
│   └── hero-banner.png
└── fonts/
    └── CustomFont.woff2
```
